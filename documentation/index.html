Get started

Request protocol: HTTP

Content type: form-data

Response Formats: JSON

Create Customer


        # Here is a curl example
        curl \
        -X GET http://ameka-art.tk/api/register

        
    


    
        # Response Message
     {
    "message": "Customer - Software Dev2 created successfully"
     }

QUERY PARAMETERS

Field Type Comments
fullname string required
email string required, unique
password string required
location string required, must be either Ibadan, Lagos, Nairobi or Mombassa
country string required, must be either Nigeria or Kenya
isCustomer boolean required, must be true
